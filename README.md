## Environment Variables

```env
MONGODB_STRING="your-mongodb-connection-string"
PORT=4000
JWT_SECRET_KEY="your-jwt-secret"
CLIENT_ID="your-google-oauth-client-id"
CLIENT_SECRET="your-google-oauth-client-secret"
```
